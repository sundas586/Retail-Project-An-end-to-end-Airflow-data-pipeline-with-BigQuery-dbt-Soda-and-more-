## Prerequisites for the project

- Docker
- Astro CLI
- Soda
- GCS account

## Astro CLI
The Astro CLI is a command-line interface for data orchestration. It allows you to get started with Apache Airflow,  It's the easiest way to get started with Apache Airflow .
